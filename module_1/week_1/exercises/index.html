DICINONÁRIO HTML

h1

Define o título mais importante.

<h1>Este é um título nível 1</h1>

h2

Define o segundo título mais importante.

<h2>Este é um título nível 2</h2>

p

Define um parágrafo.

<p>Este é um parágrafo.</p>

a

Define um hiperlink.

<a href="https://www.example.com">Este é um link</a>

img

Define uma imagem.

<img src="https://www.example.com/img.png" alt="Imagem Placeholder">

ul

Define uma lista não ordenada.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

ol

Define uma lista ordenada.

<ol>
    <li>Primeiro item</li>
    <li>Segundo item</li>
    <li>Terceiro item</li>
</ol>

li

Define um item de lista.

<ul>
    <li>Item de lista</li>
</ul>

table

Define uma tabela.

<table>
       <tr>
          <th>Cabeçalho 1</th>
          <th>Cabeçalho 2</th>
        </tr>
        <tr>
          <td>Dado 1</td>
          <td>Dado 2</td>
        </tr>
</table>

th

Define uma célula de cabeçalho em uma tabela.

<table>
       <tr>
         <th>Cabeçalho</th>
       </tr>
</table>

tr

Define uma linha em uma tabela.

<table>
       <tr>
         <td>Dado da linha</td>
       </tr>
</table>

td

Define uma célula em uma tabela.

<table>
      <tr>
         <td>Dado da célula</td>
      </tr>
</table>

div

Define uma divisão ou seção.

<div>Este é um elemento div.</div>

span

Define uma seção em um documento.

<span>Este é um elemento span.</span>

form

Define um formulário HTML para entrada do usuário.

<form action="/submit">
       <label for="name">Nome:</label>
       <input type="text" id="name" name="name">
       <input type="submit" value="Enviar">
</form>
